- Desarrollo -

Para esta práctica, descargamos un modelo 3D de un carro y lo editamos para simplificar sus detalles y dividirlo en dos partes. Posteriormente, imprimimos un molde con una impresora 3D. El proceso para fabricar el molde con fibra de vidrio comenzó aplicando una capa de resistol al molde, asegurándonos de cubrirlo completamente hasta que quedara blanco. Luego, preparamos una mezcla de resina con catalizador en proporciones iguales, ya que es importante mantener este equilibrio para que la mezcla funcione correctamente. El siguiente paso fue similar a hacer papel maché: se colocaba una capa de fibra de vidrio sobre el molde y, encima de esta, se aplicaba la mezcla de resina y catalizador. Este procedimiento se repetía al menos tres veces en ambas partes del molde para garantizar la resistencia del resultado final. Una vez que la mezcla se endureció, simplemente retiramos el molde para obtener las piezas en fibra de vidrio.

- Resultados -

Para esta práctica, descargamos un modelo 3D de un carro y lo editamos para simplificar sus detalles y dividirlo en dos partes. Posteriormente, imprimimos un molde con una impresora 3D. Debido a que nuestro modelo contaba con candados, retirar las piezas del molde resultó un poco complicado. Sin embargo, con cuidado y utilizando una espátula para hacer palanca, logramos extraerlas sin dañar el molde ni las piezas. Aunque fue un proceso laborioso, al final, el modelo salió correctamente.
